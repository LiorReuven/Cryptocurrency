# CryptoApp - Made with NextJs
Website created to provide Cryptocurrency information and statistics

Demo: [Live Demo]('') 

## Features

- Top 10 coin page 
- Details page for a specific coin
- Utilization of pre-render functionality of nextjs  
- External API calls
- Chakra-UI
- Light/Dark Mode
- Mobile support
